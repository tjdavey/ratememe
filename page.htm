About RateMeme

RateMeme is a fully-functional demonstration of several Google App Engine technologies. It was built and designed to introduce beginners to the power and ease with which basic web applications can be developed on the Google Cloud Platform.

This application was designed and implemented by Tristan Davey. The source of this entire application is available under the MIT Licence on Github.
